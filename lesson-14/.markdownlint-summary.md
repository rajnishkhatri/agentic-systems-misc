- **Count:** 20 instances
- **Pattern:** `**Bold Text:**` used for subsection headers within examples
- **Decision:** SUPPRESSED - Intentional styling choice for nested structure
- **Rationale:**

  - Bold emphasis provides visual hierarchy without breaking document outline
  - Commonly used pattern in technical documentation
- **Examples:**
  - "**Step 1: Configure**" within a learning path section
  - "**Prerequisites:**" within a tutorial overview
