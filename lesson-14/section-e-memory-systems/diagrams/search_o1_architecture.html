Search O1 Architecture

Multi-Branch Search with Reason-in-Documents Module

Architecture Overview

This diagram illustrates a sophisticated search architecture that uses multi-branch retrieval and reasoning condensation. The system generates multiple search queries in parallel, retrieves documents for each branch, and uses a Reason-in-Documents module to synthesize findings before generating a final response.
